S-CDN Statistics API 매뉴얼

S-CDN 통계 데이터(트래픽·히트율·응답 상태 등)를 REST API로 조회하는 인터페이스입니다. 단일 도메인(singledomain)과 전체 계정 합산(totaldomain) 두 가지 모드를 지원하며, HTTPS POST 방식으로 JSON 응답을 반환합니다.

인증: 요청 body의 common 객체에 id + password 또는 cloud_key_value(API Key) 포함

조회 간격: 5분 / 1시간 / 1일 — 조회 기간에 따라 선택 가능한 간격이 제한됩니다

제공 통계 17종: 전송량 · Traffic · HTTP 요청 건수 · 동시접속/방문자 · 컨텐츠 정보 · Hit/Miss Count · ISP/User Agent · 국가별 요청 · Origin 통계 · 40x/50x 컨텐츠 · HTTP Status · Origin Status · 스토리지 · 그룹별 합계 · 도메인 목록 · 에러 코드

인증 방식, 요청/응답 파라미터, 에러 코드 등 전체 내용을 확인하세요.
